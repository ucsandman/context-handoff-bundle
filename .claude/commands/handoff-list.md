List saved context handoff bundles.

## Instructions

Run the `context-handoff-bundle list` CLI command.

```bash
context-handoff-bundle list [--repo-only] [--global-only] [--limit N]
```

The command shows a table of saved bundles with: ID, title, date, quality rating, and store location.

Present the output directly. If no bundles exist, say so plainly.
